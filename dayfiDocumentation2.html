Problem with NFTs ?

There are no income production mechanisms in the current NFT ecosystem, only speculative price increases over time. As NFTs become more widely used, we expect this problem to exacerbate. Certain NFT holders, such as funds that invest in expensive/useful yield-earning NFTs for the long term, are already facing this issue. These NFT holders want to earn returns before reselling them, but they lack the motivation, time, or competence to do so. The majority of these assets are sitting in wallets, unused. To generate income, NFTs such as virtual art, gaming assets, virtual land, domain names, IP rights, and others could be rented, loaned, or mortgaged s for a modest revenue stream.
